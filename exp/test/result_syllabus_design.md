## Course Description
This course introduces students to the fundamental concepts of Python programming. It covers basic syntax, data structures, functions, libraries, debugging, and project development, aimed at enhancing studentsÆ programming skills and problem-solving abilities. Ideal for beginners or those with limited programming experience.

---

## Course Learning Objectives
By the end of the course, students will be able to:
1. Write basic Python scripts utilizing proper syntax and data types.
2. Develop problem-solving skills and algorithmic thinking for practical programming tasks.
3. Manipulate core data structures: lists, dictionaries, sets, and tuples.
4. Implement code modularity through functions and modules.
5. Leverage external libraries, particularly NumPy, pandas, and matplotlib, for data analysis.
6. Apply debugging techniques and testing methodologies.
7. Complete a capstone project demonstrating their ability to conceptualize, design, and present a comprehensive Python application.
8. Engage in self-reflection and peer feedback to continuously enhance their programming skills.

## Assessment Methods
1. **Weekly Coding Assignments (30% total)**: Weekly exercises corresponding to the topics covered in class.
2. **Midterm Exam (20%)**: Assessment of understanding fundamental concepts through practical coding problems.
3. **Capstone Project (30%)**: Group project with stages including proposal, development, and a final presentation, evaluated on code quality, functionality, and presentation skills.
4. **Participation and Engagement (10%)**: Attendance, contribution to class discussions, and peer feedback involvement.
5. **Self-Reflection Journals (10%)**: Regular written reflections on learning progress and experiences throughout the course.
