  B) variable1 = 10

print('Hello')

  A) <class 'int'>
  B) <class 'float'>

## Section 6: Control Structures

### Learning Objectives
- Describe the uses of control structures in Python.
- Implement conditional statements and loops in Python code.
- Differentiate between the types of loops and their appropriate use cases.

### Assessment Questions

**Question 1:** What is the purpose of control structures in programming?

  A) To repeat code
  B) To make decisions
  C) To create data types
  D) Both A and B

**Correct Answer:** D
**Explanation:** Control structures allow you to repeat code (loops) and make decisions (conditionals).

**Question 2:** Which of the following correctly describes an 'else' statement in Python?

  A) It executes code when the if condition is true.
  B) It executes code when the if condition is false.
  C) It can be used without an if statement.
  D) It is optional and must appear before an if statement.

**Correct Answer:** B
**Explanation:** 'else' executes code when all preceding conditions in the if-elif statements are false.

**Question 3:** How does a 'for' loop in Python generally operate?

  A) It runs indefinitely until a condition is false.
  B) It iterates over a sequence of elements.
  C) It can only iterate through lists.
  D) It requires a condition to be explicitly defined before execution.

**Correct Answer:** B
**Explanation:** A 'for' loop is used to iterate over a sequence (like a list or string) and execute a block of code for each element.

**Question 4:** What could potentially cause an infinite loop in a 'while' loop?

  A) Not updating the loop control variable.
  B) Setting a condition that is always false.
  C) Using a break statement correctly.
  D) Having a single condition in the loop.

**Correct Answer:** A
**Explanation:** Failure to update the loop control variable can lead to an infinite loop, as the condition will always evaluate to true.
